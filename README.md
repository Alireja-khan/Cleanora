# Cleanora - Professional Cleaning Services

Welcome to Cleanora, a professional cleaning service website built with HTML, CSS, and JavaScript.

## Features
- Responsive design for mobile and desktop
- Professional cleaning service showcase
- Pricing plans section
- Booking system with Google Calendar integration
- Contact information and social media links

## Installation
This is a static website that can be run directly in any browser by opening the `index.html` file.

## Deployment Options

### GitHub Pages
1. Push your code to a GitHub repository
2. Go to your repository settings
3. Navigate to the "Pages" section
4. Select your main branch as the source
5. Your site will be deployed at `https://yourusername.github.io/repository-name`

### Other Hosting Platforms
You can also deploy this static site on:
- Netlify
- Vercel
- Firebase Hosting
- Any web server that serves static files
